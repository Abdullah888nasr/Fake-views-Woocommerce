# Fake Views - Woocommerce

A lightweight WooCommerce plugin that displays a fake "live visitor" count on your product pages to create urgency and social proof.

## Description

Fake Views allows you to display a custom message on your single product pages showing a random number of "current visitors". This marketing tactic uses the scarcity principle to encourage potential customers to make a purchase decision faster.

## Features

- **Customizable Range**: Set a minimum and maximum number for the random visitor count generation.
- **Position Control**: Choose where to display the text on the single product page:
  - Before the "Add to Cart" button
  - After the "Add to Cart" button
- **Simple Configuration**: Easy-to-use settings panel in the WordPress admin dashboard.

## Installation

1. Upload the `fake-views` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Fake Views Settings** to configure the plugin.

## Configuration

Navigate to **Settings > Fake Views Settings**.

- **Low Range**: The minimum number of visitors to show.
- **High Range**: The maximum number of visitors to show.
- **Position**: Select where the visitor count should appear (Before or After the Add to Cart button).
